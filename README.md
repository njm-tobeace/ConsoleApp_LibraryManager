#  What kind of program is LibraryManager?  

LibraryManager is a console program for creating and managing books. It features an interactive menu that makes it easy to use the app. It's just a small application, but one that can be continuously expanded. 

#  What is an app like this used for?  

My application itself is quite simple. A fully developed application, as the name suggests, will be capable of managing an entire library. It stores important information and can retrieve it at any time. 

#  What can my application do in its current state?  

My application is currently able to add books (title, content, and ID), delete books, mark them as checked out, mark them as returned, and retrieve all saved information.
